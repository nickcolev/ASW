ASW
Android StopWatch

Purpose
A simple chronometer.
Installation
Install from Google Play as any other application.
Usage
Start/StopTouch the timer or volume key
ResetLong-touch the timer (about 1 second) or volume key
TagTouch above the timer or use the menu
LapTouch below the timer
Log
If the timer is stopped, and a tag provided — the measured time shall be saved. To see the time log — use the menu. From the log screen, the log can be emailed for further analysis.
Precision
The time is measured in tenth of a second.
Privacy
No private data is collected by the application.
